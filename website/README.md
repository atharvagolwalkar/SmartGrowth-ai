# SmartGrowth AI Frontend

Beautiful Next.js dashboard for customer intelligence, churn prediction, demand forecasting, and NLP-powered insights.

## Features

- **📊 Overview Dashboard** - Real-time KPIs and customer analytics
- **👥 Customer Analysis** - Individual customer profiles with churn risk assessment
- **📈 Demand Forecast** - Multi-model forecasting (Prophet, N-BEATS, ARIMA)
- **🚨 High Risk Customers** - Identify and prioritize at-risk accounts
- **⚡ Batch Prediction** - Scale churn predictions across your customer base
- **🧠 NLP Insights** - Semantic search and sentiment analysis of feedback

## Quick Start

```bash
cd website
npm install
npm run dev
```

Visit http://localhost:3000

## Environment

Create `.env.local`:
```
NEXT_PUBLIC_API_URL=https://handhelds-cooling-gale-consistently.trycloudflare.com
```

## Deploy

Deploy to Vercel for free with:
```bash
vercel deploy
```
